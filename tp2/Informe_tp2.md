## Objetivo  
Diseñar e implementar un sistema multi-capa para el procesamiento de datos externos, integrando el consumo de servicios API REST mediante Python con el procesamiento de bajo nivel en C y Ensamblador (x86/ARM).
Se busca profundizar en el uso del Stack y las convenciones de llamada (Calling Conventions) para el pasaje de parámetros y retorno de resultados entre lenguajes de alto y bajo nivel.
